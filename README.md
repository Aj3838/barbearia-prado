# Barbearia Prado – Website Demo

Este projeto é uma **landing page desenvolvida em código** como demonstração de site para uma barbearia localizada na região de Taboão da Serra.

O objetivo do projeto foi criar um site moderno e simples para apresentar a barbearia, seus serviços e facilitar o contato com clientes. O layout foi pensado para ter uma identidade visual forte, navegação fácil e uma experiência agradável tanto no computador quanto no celular.

## Sobre o projeto

Diferente de outros projetos feitos em construtores de site, este foi **desenvolvido manualmente**, utilizando código para estruturar e estilizar toda a página.

O projeto simula como seria o site oficial da barbearia, servindo como demonstração de presença digital para negócios locais.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## Funcionalidades

* Apresentação da barbearia
* Seção de serviços
* Layout moderno e responsivo
* Estrutura de landing page para negócios locais

Este projeto faz parte do meu portfólio de desenvolvimento web e demonstra a criação de um site completo utilizando código, focado em pequenos negócios locais.
